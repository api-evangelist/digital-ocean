DigitalOcean introduced Server-Side Tools for its Inference Engine in Public Preview, enabling AI models to search the web, read data, call systems, and take action within a single inference request. The feature covers web search, web fetch, Knowledge Bases, MCP servers, and support for Anthropic and OpenAI tool conventions, all accessible through existing DigitalOcean credentials without additional infrastructure setup.
