## APIs

### Digital Ocean API

The DigitalOcean API lets you programmatically manage your Droplets and other resources using conventional HTTP requests. Any action that you can perform through the DigitalOcean Control Panel (except for creating personal access tokens) can also be performed with the API. Resources include Droplets, Kubernetes, Apps, Databases, Spaces, Volumes, Load Balancers, Networking, Functions, Container Registry, Monitoring, and Billing.
